# Cyclistic Bike-Share Analysis

## Project Overview
This project analyzes 12 months of Cyclistic bike-share data (over 5.5 million rows) to understand behavioral differences between casual riders and annual members, with the goal of identifying opportunities to convert casual users into members.

## Business Question
How do casual riders differ from annual members in usage patterns, and how can these insights inform membership conversion strategies?

## Tools & Technologies Used
- Python (pandas, matplotlib)
- Power BI (PowerQuery, DAX, Visuals, Slicers)
- Visual Studio Code, GitHub

## Key Insights
- Casual riders take significantly longer trips, indicating leisure-focused usage
- Annual members dominate weekday commute hours
- Casual usage increases on weekends and afternoons
- Weekend-focused promotions may increase member conversion

## Data Preparation
- Started with data cleaning and visualization on data for 1 month to prototype and test my approach, then scaled gradually to incorporate 12 months of cyclistic data.
- Cleaned and merged 12 monthly CSV files
- Removed invalid ride durations
- Generated analysis friendly features such as weekday, hour, and day type

## Conclusion
This analysis highlights clear behavioral differences between rider types and suggests targeted marketing strategies to convert high-engagement casual riders into annual members.
